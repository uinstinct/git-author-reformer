<objective>
Implement the drop-coauthor half of the rewrite engine following strict TDD: write failing tests first (RED), then implement pub fn drop_coauthor in src/git/rewrite.rs (GREEN). Closes requirements DROP-02 (remove selected co-author from all Co-authored-by trailers, case-insensitive email key, remove all duplicate occurrences within a single commit) and DROP-03 (all other Co-authored-by entries, commit message body, other trailers, tree, timestamps, author, and committer preserved byte-for-byte).

The engine reuses the topological walk, OID-map, ref/tag/HEAD update pass machinery from Plan 02-02. The decision logic differs: a commit needs rewriting if its commit message contains at least one Co-authored-by line whose email (case-insensitive) matches the target email OR any of its parents were rewritten. The message rewrite logic is delegated to a private pure function drop_coauthor_from_message that operates on the raw message string.

Purpose: drop-coauthor is the second user-facing operation. With DROP done, Phase 2 completes both engine operations, leaving Phase 3 to wire them to the TUI.

Output: pub fn drop_coauthor in src/git/rewrite.rs plus integration tests proving correctness across single trailer, multiple trailers, case-insensitive match, duplicate occurrences within one commit, and byte-identical preservation of all unaffected metadata.
</objective>
