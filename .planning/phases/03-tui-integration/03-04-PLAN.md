<objective>
Implement the Drop co-author flow up to Screen::Preview: fuzzy-filterable co-author list (DROP-01). Selecting a co-author transitions directly to `Screen::Preview(PendingOp::Drop { ... })` — there is no second screen because drop only needs the target email (the co-author identity itself).

Purpose: DROP-01 mirrors RENAME-01 but enumerates Co-authored-by trailer identities instead of primary authors. The drop flow is shorter than the rename flow because no replacement values are needed — the operation is removal, not substitution.

Output: From the main menu, pressing Enter on "Drop a co-author" loads the co-author list; typing filters it; selecting one transitions to the Preview placeholder (Plan 03-05 implements the warnings + confirmation render).
</objective>
